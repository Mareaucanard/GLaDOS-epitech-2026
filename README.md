# Build guide:
## On linux:
### Requirements:
* Git (or any other way to clone the repo)
* Docker
* Make
### Build:
```bash
git clone https://github.com/Mareaucanard/GLaDOS-epitech-2026.git
cd GLaDOS-epitech-2026
make
```
That's it!
You can now run the program with `./glados`
