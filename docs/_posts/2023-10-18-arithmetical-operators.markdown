### The GLaDOS language supports the following arithmetical operators:

| Operator | Description |
|:--------:|:------------|
| `+`    | Adition |
| `-`     | Substraction |
| `*`    | Multiplication |
| `/`    | Division |
| `%`    | Modulo |
